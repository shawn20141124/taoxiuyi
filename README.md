## 🔬 技术看点（终极版新增亮点）

`DadChallenge.cpp` 可以说是 **Windows 恶意技术大杂烩**，集成了以下高级机制：

| 技术点 | 功能与原理 | 实际效果 |
|:---|:---|:---|
| **UAC 绕过 (fodhelper)** | 利用注册表劫持 `ms-settings` 命令，以管理员权限执行 | 无弹窗提权，悄无声息 |
| **计划任务自启动** | 创建以 `SYSTEM` 账户运行的开机计划任务 | 重启后自动复活，杀不掉 |
| **双进程守护** | 全局互斥体 + 守护者进程相互监视 | 一个被结束，另一个立刻拉起 |
| **自我复制并删除** | 复制自身到 `System32` 下并隐藏，原文件自删 | 伪装系统文件，难以清理 |
| **反调试/反虚拟机** | 检测调试器、VirtualBox、VMware 存在痕迹 | 发现分析环境则自我销毁 |
| **状态持久化** | 注册表 + ADS 备用流双重记录状态 | 上次非正常退出，下次直接蓝屏 |
| **进程伪装** | 修改 PEB 中的路径和命令行，伪装为 `svchost.exe` | 任务管理器里混水摸鱼 |
| **关键进程保护** | `RtlSetProcessIsCritical` | 结束进程立刻蓝屏 |
| **手动蓝屏** | `NtRaiseHardError` 主动触发 | 不讲武德的一键重启 |
| **🆕 AES-256 文件加密** | 使用 Windows BCrypt API，生成随机密钥/IV，**密钥不保存**，加密后追加魔数标记 | 用户文件永久无法恢复 |
| **🆕 目标文件扩展名过滤** | 加密 `doc`、`pdf`、`jpg`、`cpp`、`zip` 等 30+ 种常见文件类型 | 你的所有数据无一幸免 |
| **🆕 跳过系统目录** | 遍历磁盘时跳过 `Windows`、`Program Files` 等目录，避免影响系统运行 | 专门破坏你的个人数据 |
| **🆕 删除关键系统文件** | 删除 `BCD`、`winload.exe`、`ntoskrnl.exe` 等引导文件 | 系统彻底瘫痪，无法启动 |
| **🆕 加密防重复机制** | 注册表记录“已加密”状态，避免重复加密或重启后再次执行 | 一次性破坏，干净利落 |

## 🛠 编译 & 运行（超详细，小白也能看懂）

### 1. 你需要准备的

- 一台安装了 **Windows 10/11** 的虚拟机（Vmware 或 VirtualBox）
- 虚拟机里装好 **Visual Studio 2019/2022**（必须勾选“使用C++的桌面开发”）
- 虚拟机拍好快照（**非常重要！**）

1. 打开 **“Developer PowerShell for VS”**（在开始菜单搜索）
