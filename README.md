## 🔬 技术看点（如果你关心代码本身）

`叫爸爸程序.cpp` 里用到了一些有意思的 Windows 底层技术：

- **RtlSetProcessIsCritical** — 将进程设置为系统关键进程，终止后导致蓝屏
- **PEB 篡改** — 伪装进程路径为 `svchost.exe`，在任务管理器中隐藏行迹
- **守护线程 + 蓝屏保护** — 主进程被杀？那就同归于尽。
- **双注册表自启动** — 在 HKLM 和 HKCU 的 `Run` 键实现双重保障
- **NtRaiseHardError** — 手动触发系统蓝屏

这些技术本身是双刃剑：了解它，是为了更好地理解操作系统；用它来整人，请务必在**知情同意**的前提下。
