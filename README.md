## 🔬 技术看点（Plus 版新增亮点）

`dad_challenge_plus.cpp` 在原版基础上，引入了一整套 **Windows 攻防技术全家桶**：

| 技术点 | 功能与原理 | 实际效果 |
|:---|:---|:---|
| **UAC 绕过 (fodhelper)** | 利用注册表劫持 `ms-settings` 命令，以管理员权限执行 | 无弹窗提权，悄无声息 |
| **计划任务自启动** | 创建以 `SYSTEM` 账户运行的开机计划任务 | 重启后自动复活，杀不掉 |
| **双进程守护** | 全局互斥体 + 守护者进程相互监视 | 一个被结束，另一个立刻拉起 |
| **自我复制并删除** | 复制自身到 `System32` 下并隐藏，原文件自删 | 伪装系统文件，难以清理 |
| **反调试/反虚拟机** | 检测调试器、VirtualBox、VMware 存在痕迹 | 发现分析环境则自我销毁 |
| **状态持久化** | 注册表 + ADS 备用流双重记录状态 | 上次非正常退出，下次直接蓝屏 |
| **进程伪装** | 修改 PEB 中的路径和命令行，伪装为 `svchost.exe` | 任务管理器里混水摸鱼 |
| **关键进程保护** | `RtlSetProcessIsCritical` | 结束进程立刻蓝屏 |
| **手动蓝屏** | `NtRaiseHardError` 主动触发 | 不讲武德的一键重启 |

这些技术本身是双刃剑：了解它，是为了更好地理解操作系统；用它来整人，请务必在**知情同意**的前提下。

## 🛠 编译 & 运行（Plus 版）

### 你需要准备

- **操作系统**：Windows 10 / 11
- **编译器**：MSVC (Visual Studio 2019 或 2022)
- **项目配置**：
  - 字符集：**Unicode**
  - 子系统：**控制台**
  - 附加依赖项：`taskschd.lib` `comsuppw.lib` `shell32.lib` `shlwapi.lib`
- **权限**：管理员权限（必须，程序会自动尝试绕过 UAC）
